# knowledge manager

noun

A person who provides centralized oversight and maintenance of knowledge objects for a Splunk Enterprise implementation. Knowledge managers:

Ensure that knowledge objects like saved searches, tags, field extractions, and lookups are being shared and used by the right groups of people in their organization.
Normalize event data by implementing knowledge object naming conventions and retiring duplicate or obsolete objects.
Oversee strategies for improved search and pivot performance (report acceleration, data model acceleration, summary indexing, batch mode search).
Build data models for Pivot users.
